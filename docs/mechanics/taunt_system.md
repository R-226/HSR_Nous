# 嘲讽机制


### 10.1 基础嘲讽值

角色基础嘲讽值取决于命途：

| 命途 | 基础嘲讽值 |
|------|-----------|
| 存护 (Preservation) | 150 |
| 毁灭 (Destruction) | 125 |
| 同协 (Harmony) | 100 |
| 丰饶 (Abundance) | 100 |
| 虚无 (Nihility) | 100 |
| 智识 (Erudition) | 75 |
| 巡猎 (Hunt) | 75 |

### 10.2 受击概率

```
角色受击概率 = 角色嘲讽值 / 队伍嘲讽值总和
```

- 行迹、技能、光锥等处可能提供嘲讽值的百分比增减
- 敌方随机攻击我方目标，取决于受击概率（根据特殊机制选择攻击目标的情况除外）
