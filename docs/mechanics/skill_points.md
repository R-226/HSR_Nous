# 战技点机制


### 6.1 战技点上限

- 全队共享，默认上限 **5** 点
- 部分角色天赋可提高上限（如花火天赋可将上限提高至 **7** 点）

### 6.2 战技点获取与消耗

#### 常规规则

| 行为 | 战技点变化 |
|------|-----------|
| 普攻 | +1 |
| 战技 | -1 |
| 终结技 | 0 |
| 部分光锥/行迹效果 | 视具体效果 |

#### 特殊角色/机制

| 类型 | 说明 | 示例 |
|------|------|------|
| 战技不消耗战技点 | 释放战技时无视战技点消耗 | 如遐蝶、万敌战技 |
| 战技消耗多个战技点 | 单次战技消耗大于 1 点 | 如 Archer 战技消耗 2 点 |
| 普攻不恢复战技点 | 强化普攻等特殊普攻不产生战技点 | 如刃强化普攻 |
| 终结技回复战技点 | 释放终结技时额外回复战技点 | 如花火终结技 |
| 光锥/行迹额外回复 | 通过装备或被动额外获取战技点 | 如部分光锥特效 |
