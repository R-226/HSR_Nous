# 基础属性


### 1.1 角色属性

| 属性 | 说明 | 计算方式 |
|------|------|---------|
| 生命值 (HP) | 基础生命 + 遗器/光锥加成 | 白值 × (1 + 百分比加成) + 固定加成 |
| 攻击力 (ATK) | 基础攻击 + 遗器/光锥加成 | 白值 × (1 + 百分比加成) + 固定加成 |
| 防御力 (DEF) | 基础防御 + 遗器/光锥加成 | 白值 × (1 + 百分比加成) + 固定加成 |
| 速度 (SPD) | 决定行动频率 | 基础速度 × (1 + 百分比加成) + 固定加成 |
| 暴击率 | 触发暴击的概率 | 基础 5% + 加成 |
| 暴击伤害 | 暴击时的伤害倍率 | 基础 50% + 加成 |
| 击破特攻 | 击破时额外伤害 | 基础 0% + 加成 |
| 治疗量加成 | 造成治疗时额外的加成 | 基础 0% + 加成 |
| 能量上限 | 角色通常情况下可累积的能量值的上限 | 固定值 |
| 能量恢复效率 | 获取能量的倍率 | 基础 100% + 加成 |
| 效果命中 | 施加 debuff 的成功概率 | 基础 0% + 加成 |
| 效果抵抗 | 抵抗 debuff 的概率 | 基础 0% + 加成 |
| 欢愉度 | 影响欢愉伤害的基础属性，能提高造成的欢愉伤害倍率 | 基础 0% + 加成 |

#### 基础属性计算

```
攻击白值 = 角色基础攻击 + 光锥基础攻击
攻击 = 攻击白值 × [1 + (攻击% + buff攻击%)] + 额外固定攻击

生命白值 = 角色基础生命 + 光锥基础生命
生命 = 生命白值 × [1 + (生命% + buff生命%)] + 额外固定生命

防御白值 = 角色基础防御 + 光锥基础防御
防御 = 防御白值 × [1 + (防御% + buff防御%)] + 额外固定防御

基础速度 = 角色基础速度 + 光锥/行迹/星魂等提供的基础速度
速度 = 基础速度 × (1 + 速度% + buff速度%) + 额外固定速度
```

### 1.2 敌人属性

敌人额外拥有：
- **韧性 (Toughness)**：被攻击时减少，归零时触发弱点击破
- **弱点属性**：对应属性攻击可削减韧性
- **抗性**：对特定属性伤害的减免

#### 弱点与抗性的关系

- 敌人天生拥有的弱点对应属性抗性为 **0%**，非弱点属性抗性为 **20%**
- **弱点与抗性非强关联**：添加弱点不会自动降低对应属性抗性
  - 如奇物"朋克洛德精神"添加弱点但不降低抗性
  - 银狼战技同时添加弱点并降低抗性，这是两个独立效果
- 部分敌人额外拥有 **+20%** 的属性抗性，叠加后最高可达 **40%**

### 1.3 技能与打击方式

己方角色拥有三类技能：

| 技能类型 | 常规资源消耗 | 特殊机制示例 |
|---------|-------------|-------------|
| 普攻 (Basic ATK) | 通常回复 1 战技点 | 如刃强化普攻不回复战技点 |
| 战技 (Skill) | 通常消耗 1 战技点 | 如遐蝶、万敌战技不消耗；如 Archer 战技消耗 2 点 |
| 终结技 (Ultimate) | 消耗满能量，立即插队释放 | 如花火终结技可回复战技点 |

技能打击方式：

| 打击方式 | 说明 | 示例 |
|---------|------|------|
| 单体 | 攻击单个目标 | 如丹恒战技 |
| 扩散 | 攻击主目标及相邻目标 | 如姬子战技 |
| 弹射 | 多次随机攻击，可重复命中同一目标 | 如瓦尔特战技 |
| 群体 | 攻击全体敌人 | 如姬子终结技 |

技能效果类型：

| 效果类型 | 说明 | 示例 |
|---------|------|------|
| 回复 | 恢复生命/能量/解除负面状态 | 如丰饶角色治疗 |
| 防御 | 添加护盾/防御增益 | 如存护角色护盾 |
| 辅助 | 施加增益状态 | 如同协角色 buff |
| 妨害 | 施加负面状态/解除敌人增益 | 如虚无角色 debuff |
| 单攻/群攻/扩散/弹射 | 造成伤害 | 如输出角色技能 |

> 若技能效果包含多种类型，UI 上只会显示其中一种主要类型。

### 1.4 记忆命途与忆灵（召唤物）

记忆命途角色的核心特征是**召唤忆灵**。忆灵是召唤物的一个子类。

#### 名词定义

| 术语 | 定义 |
|------|------|
| **忆师** | 能够召唤忆灵的角色 |
| **忆灵** | 由记忆命途角色召唤出的独立单位，属于召唤物的子集 |

#### 忆灵机制

1. **召唤物关系**：忆灵 ⊂ 召唤物。对召唤物生效的效果对忆灵也生效；仅对忆灵生效的效果对非忆灵召唤物无效。
2. **独立单位**：忆灵拥有独立的血条和行动序列，可被技能选择为目标，也可被敌方攻击。
3. **属性继承**：忆灵初始属性继承忆师的**战斗外面板**；有特殊说明的优先按特殊规则继承。
   - 例：开拓者·记忆的忆灵"迷迷"基础速度固定为 130，与忆师速度无关；生命值按一定比例继承忆师生命值并享有额外固定加成；其余属性无特殊说明时与忆师战斗外面板一致。
4. **面板独立**：战斗内忆师与忆灵的面板变化**互不干扰**，获得的单体增益**不共享**。
   - 例：遗器套装【奇想蕉乐园】的 16% 暴击伤害为战斗外面板增益，忆师和忆灵均可获得；32% 暴击伤害为战斗内对装备者生效的增益，仅忆师获得。
5. **独立技能组**：忆灵拥有单独的技能组，部分在行动值归零时自动施放，部分由玩家选择目标施放。具体机制因角色而异。
6. **能量恢复**：忆灵行动施放技能时，会为**忆师**恢复能量。
